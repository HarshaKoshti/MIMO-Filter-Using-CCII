# MIMO Filter Using Current Conveyor (CCII)

## Project Overview

This project presents the design, simulation, and PCB implementation of a MIMO (Multiple Input Multiple Output) Filter using the Current Conveyor II (CCII) architecture in Cadence OrCAD X.

The objective of this project is to analyze the frequency response of active filters and obtain:

- Low Pass Filter (LPF)
- Band Pass Filter (BPF)
- High Pass Filter (HPF)

using Current Conveyor based analog design.

---

## Software Used

- Cadence OrCAD X
- OrCAD Capture
- PSpice Simulator
- Allegro PCB Editor

---

## Design Flow

1. Circuit Design
2. Simulation using PSpice
3. AC Frequency Analysis
4. PCB Layout Design
5. Design Rule Check (DRC)
